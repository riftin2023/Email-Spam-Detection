# 📧 Spam Detection using LSTM & NLP

## 🚀 Overview

This project builds a **Spam Email/SMS Classifier** using **Natural Language Processing (NLP)** and a **Deep Learning (LSTM) model**.
The model classifies messages as **Spam** or **Ham (Non-Spam)**.

---

## 🧠 Features

* Text preprocessing (cleaning, stopword removal, punctuation removal)
* Tokenization and sequence padding
* LSTM-based deep learning model
* Early stopping & learning rate optimization
* Visualization using WordCloud and accuracy graphs
* High accuracy classification (~95%+)

---

## 🛠️ Tech Stack

* Python 🐍
* TensorFlow / Keras
* NumPy & Pandas
* Matplotlib & Seaborn
* NLTK (Natural Language Toolkit)

---
## 🔄 Workflow

1. Data Cleaning
2. Text Preprocessing
3. Tokenization
4. Sequence Padding
5. Model Building (LSTM)
6. Training & Evaluation
7. Visualization

---

## 🤖 Model Architecture

* Embedding Layer
* LSTM Layer
* Dense Layer (ReLU)
* Output Layer (Sigmoid)

---

## 📊 Results

* Training Accuracy: ~97–98%
* Validation Accuracy: ~95–96%
* Test Accuracy: ~95%+

📉 The model shows **good generalization with minimal overfitting**.
